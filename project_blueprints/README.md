# project_blueprints/

Per-project design plans. Blueprints are **documentation** — they describe how to
transfer and apply the shared playbook, not implementation code.

| Blueprint | Scope | Posture |
| --- | --- | --- |
| [project_money.md](project_money.md) | Multi-asset quant strategy research (equities, ETFs, options, macro, factors) | Research-only; bootstrap |

Read a blueprint alongside [../CLAUDE.md](../CLAUDE.md) and
[../docs/safety_policy.md](../docs/safety_policy.md). A blueprint never authorizes
leaving the current phase — that requires explicit user approval and passing the
gates in [../docs/promotion_policy.md](../docs/promotion_policy.md).
